+ Added: You can now stack limit packs for players:
packs:
  # If enabled, all packs the player has, will be merged and the limits stacked.
  # Basically the limit of each pack that the player has, will be added on top of each other.
  # If you change this value, you may want to update already configured packs accordingly.
  stack: false

The option needs to be added manually to the player-limits file in this case. See default file: https://github.com/IncrediblePlugins/Lands/blob/main/player-limits.yml
